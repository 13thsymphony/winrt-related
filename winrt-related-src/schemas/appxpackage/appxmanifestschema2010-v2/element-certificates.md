# Certificates




Declares a package extensibility point of type **windows.certificates**. The app requires one or more certificates from the specified certificate stores.

## Syntax

``` syntax
<Certificates>

  <!-- Child elements -->
  Certificate{0,100},
  TrustFlags?,
  SelectionCriteria?

</Certificates>
```

### Key

`?`   optional (zero or one)
`{}`   specific range of occurrences
